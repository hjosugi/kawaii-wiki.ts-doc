# Development & Contribution

The goal of this project is to collaboratively nurture high-quality free packages, enabling users, developers, people, and AI to increase knowledge and make connections more easily.

## Local Development

Start with `bun install` and `bun run dev`. Before and after changes, run `bun run lint`, `bun run typecheck`, `bun run test`, and `bun run build`. For UI changes, verify on narrow screens, with keyboard operation, and in both Japanese and English.

## Issue

Include the expected behavior, actual behavior, reproduction steps, URL, screen width, and logs. Do not include confidential information, access tokens, or personal data.

## Pull Request

Provide the reason for the change, scope, test results, compatibility, and screenshots. Prioritize small, reviewable changes, and include upgrade instructions for data migrations or API changes.
