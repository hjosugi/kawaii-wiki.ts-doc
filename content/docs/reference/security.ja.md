# セキュリティポリシー

## サポート対象バージョン

最新の1.xリリースはセキュリティ修正を受け取ります。1.0.0が公開された後は、1.0未満のリリースはサポートされません。

## 脆弱性の報告

このリポジトリの**Security → Report a vulnerability**のプライベート報告フォームを使用してください。エクスプロイトの詳細、秘密情報、個人データ、または未修正の脆弱性を含む公開のIssueは開かないでください。

影響を受けるバージョン、展開の前提条件、再現手順、影響範囲、および推奨される修正方法を含めてください。メンテナは完全な報告をできるだけ早く確認し、非公開で検証と修正の調整を行い、匿名希望がない限り報告者にクレジットを付与します。

展開の強化には、ユニークな`JWT_SECRET`、HTTPS、最新のイミュータブルイメージタグ、定期的にテストされたバックアップを使用し、展開で明示的に必要とされない限りプライベートWebhookターゲットは無効にしてください。
