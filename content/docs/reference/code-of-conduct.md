# Code of conduct

We are committed to a welcoming, harassment-free project for everyone.
Contributors must communicate respectfully, assume good intent while evaluating
technical claims on evidence, and avoid harassment, discrimination, threats,
doxxing, or sexualized conduct.

Maintainers may edit or remove abusive content and temporarily or permanently
restrict participation when necessary to protect the community. Report conduct
concerns privately through the repository owner's GitHub profile contact. Do
not use the public issue tracker for sensitive personal reports.
