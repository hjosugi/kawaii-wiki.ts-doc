# 行動規範

私たちは、すべての人にとって歓迎される、嫌がらせのないプロジェクトを目指しています。
貢献者は敬意を持ってコミュニケーションを行い、技術的な主張を評価する際には根拠に基づいて善意を前提とし、嫌がらせ、差別、脅迫、個人情報の暴露（ドキシング）、性的な行為を避けなければなりません。

メンテナは、コミュニティを保護するために必要に応じて、悪質な内容を編集または削除し、一時的または恒久的に参加を制限することがあります。
行動に関する懸念は、リポジトリ所有者のGitHubプロフィールの連絡先を通じて非公開で報告してください。
個人的な敏感な報告には、公開のイシュートラッカーを使用しないでください。
