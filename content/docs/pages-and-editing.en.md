# Creating and Editing Pages

## Viewing and Editing

The viewing screen displays "Viewing," while the editing screen shows "Editing" or "Creating New Page." You can switch to editing from the "Edit" button on the viewing screen, and other operations are grouped under the "More" menu.

## Editor

- **Visual**: Edit regular text in a form close to its final appearance
- **Markdown**: Edit Markdown directly
- **Collaborative**: Multiple users editing simultaneously

You can insert headings, bullet lists, links, tables, code, quotes, and images from the toolbar. The save status shows "Unsaved Changes," "Saving," or "Saved."

## Page Information

You can set the title, path, status, labels, space, language, review date, scheduled publication, sidebar order, icon, and cover image. The normal viewing screen displays only important status and update time, so detailed information for editing does not clutter the main content.

## History and Restoration

A revision is created with each save. You can compare differences and restore the desired version from "More → History." Deleted pages are moved to the trash and can be restored or permanently deleted from the admin panel.
